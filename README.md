# CRUD de Funcionários

Fazendo um CRUD por semana até eu me sentir confiante que sei fazer CRUDs

### STS
- git clone https://github.com/analudias/employee-crud.git
- Importe o projeto no STS
- Execute o projeto

### Postman
- Crie novas requests
  - POST New employee
    - {{host}}/employees/
  - PUT Update employee
    - {{host}}/employees/1
  - DEL Delete employee
    - {{host}}/employees/1
  - GET Employee by id
    - {{host}}/employee/1
  - GET All employees
    - {{host}}/employee
